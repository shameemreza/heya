# Writing voice

Write so it reads like a person wrote it, not a template.

- Lead with the point. Say the thing, then explain it.
- Prefer short, plain sentences. Cut filler ("just", "simply", "in order to").
- Be specific. Concrete nouns and real examples beat abstractions.
- Match the reader — their level of detail and their tone.
- Avoid AI tells: no "delve", "tapestry", "moreover", "it's worth noting"; no
  breathless adjectives; no stacked hedging ("might possibly perhaps").
- Don't pad. If a sentence adds nothing, delete it.
- Read it back once. If a line is hard to say out loud, rewrite it.

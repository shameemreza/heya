# Banned AI words, phrases, and patterns

If a word or phrase appears here, do not use it. Use the alternative, or remove
it. This is the source of truth for what makes writing sound like a machine. The
goal is plain, human prose, so read it back and cut anything that reads like a
template.

## Transition words

| Banned | Use instead |
|---|---|
| Furthermore | And, On top of that, or remove |
| Additionally | Also, Plus |
| Moreover | What's more, or remove |
| However | But, That said, Still |
| Therefore | So, This means |
| Consequently | Because of this, This leads to |
| Subsequently | Next, After that, Then |
| Nonetheless | Even so, Still, Despite this |
| Hence | So, That's why |
| Thus | So, This means |
| In order to | To |
| Prior to | Before |
| Regarding, Pertaining to | About |
| Moving forward, Going forward | From now on, Next, or remove |

## Hedge and filler phrases

| Banned | Use instead |
|---|---|
| It is important to note | State the fact directly |
| It's worth noting (that) | Remove, just state it |
| It's worth mentioning | Remove, just mention it |
| That being said | That said, Still, But |
| With that in mind | With that, Given this |
| At the end of the day | Remove, state it directly |
| In a nutshell, In short (as filler) | Remove if it adds nothing |
| The bottom line is | Remove, just state it |
| At its core | Basically, Really, or remove |
| To put it simply | Remove, just say it simply |
| In conclusion | So, or remove the line |
| As per my previous message | Remove, do not remind people of a prior message |
| Needless to say | Remove |
| Rest assured | Remove, be concrete instead |

## Overused AI words

Avoid these unless they are genuinely the right word: delve, leverage, robust,
seamless, tapestry, realm, landscape, navigate (as a metaphor), unlock, harness,
elevate, empower, foster, facilitate, utilize (use "use"), commence (use
"start"), endeavor, plethora, myriad, crucial, pivotal, vibrant, bustling,
testament, beacon, treasure trove, game-changer, cutting-edge, state-of-the-art,
ever-evolving, fast-paced, dive in, deep dive.

## Patterns to avoid

- Stacked hedging: "might possibly perhaps", "it could potentially maybe".
- Em dashes used for dramatic pauses. Use a comma, a period, or a new sentence.
- The "It's not just X, it's Y" construction.
- Rule-of-three lists for rhythm when two items or one would do.
- Opening with "In today's world" or "In the world of".
- Breathless adjectives stacked on a plain noun.
- Closing with a summary that repeats what you just said.
- Emoji, unless the user used them first.

Re-read this before you finish a piece of writing. Cut every match.
